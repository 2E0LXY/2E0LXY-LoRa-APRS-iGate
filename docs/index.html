Install 2E0LXY LoRa APRS iGate

This installer supports Heltec WiFi LoRa 32 V3.2, Heltec WiFi LoRa 32 V4 and LilyGO TTGO LoRa32 V2.1. Select the exact board below; no drivers or extra software are normally required.

Connect USB

Connect one supported Heltec or LilyGO board directly to this computer.

Select exact board

V3.2 and V4 are both ESP32-S3 but have different hardware. Choose the matching installer.

Install firmware

Allow the write and verification to finish completely.

Heltec V3.2 or LilyGO V2.1 Install V3.2 / LilyGO Use desktop Chrome or Edge with Web Serial enabled. Web Serial installation must run over HTTPS.
Heltec WiFi LoRa 32 V4 only Install Heltec V4 Use desktop Chrome or Edge with Web Serial enabled. Web Serial installation must run over HTTPS.

Configuration may be erased by a full installation. Never select V4 for a V3.2 board or vice versa.
